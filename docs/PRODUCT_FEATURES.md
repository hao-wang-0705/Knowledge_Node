# Nexus - AI-Native 节点式知识操作系统

> **版本**: v3.6  
> **更新日期**: 2026-03-09

---

## 📋 产品概述

**Nexus** 是一款 AI 原生的节点式知识操作系统，通过树形结构组织知识，结合双螺旋标签体系和 AI 增强能力，帮助用户高效管理、检索和利用个人知识库。

### 核心理念

- **节点即知识原子**：一切信息以节点为基本单元，支持无限嵌套
- **标签即语义增强**：通过超级标签赋予节点结构化属性
- **AI 即智能助手**：AI 深度集成，从捕获、提取到聚合全流程赋能

## 🎯 核心功能模块

### 1. 树形节点系统 (Tree Node System)

| 功能 | 说明 |
|------|------|
| **无限嵌套** | 节点可无限层级嵌套，构建任意深度的知识树 |
| **拖拽缩进** | Tab/Shift+Tab 快捷键实现节点层级调整 |
| **折叠展开** | 一键折叠/展开子树，聚焦当前工作区域 |
| **批量操作** | 支持节点批量创建、更新、删除 |
| **引用系统** | 独立引用管理，支持节点间双向链接 |

**节点类型：**

| 类型 | 说明 | 图标 |
|------|------|------|
| `text` | 普通文本节点 | 默认 |
| `heading` | 标题节点 | 📌 |
| `todo` | 待办事项节点 | ☑️ |
| `command` | AI 指令节点 | 🤖 |
| `daily` | 每日笔记节点 | 📅 |
| `search` | 搜索节点 | 🔍 |

---

### 2. 超级标签体系 (Supertag System)

超级标签为节点赋予结构化属性，实现类 Schema 的数据管理能力。

#### 字段类型

| 字段类型 | 说明 | 示例 |
|----------|------|------|
| `text` | 文本字段 | 作者、备注 |
| `number` | 数字字段 | 评分、金额 |
| `date` | 日期字段 | 截止日期、创建时间 |
| `select` | 单选字段 | 状态、优先级 |
| `multi-select` | 多选字段 | 标签、分类 |
| `reference` | 节点引用字段 | 关联任务、相关项目 |
| `ai_text` | AI 生成文本字段 | 自动摘要 |
| `ai_select` | AI 生成选择字段 | 自动分类 |

#### AI 字段预设

| 预设类型 | 说明 |
|----------|------|
| `extraction` | 信息抽取型 - 从内容中提取关键信息 |
| `summarization` | 总结重写型 - 生成 TL;DR 或格式化重写 |
| `classification` | 自动分类型 - 根据内容特征自动分类 |

#### 模版系统

超级标签支持定义默认内容模版，应用标签时自动生成子节点结构。

---

### 3. 双模式导航 (Dual Mode Navigation)

#### 📅 日历模式 (Daily Notes)

自动生成年/月/周/日层级的日记节点树：

```
📅 2026
 └── 📅 2026-03
     └── 📅 Week 10
         └── 📅 2026-03-09 Monday
```

#### 📓 笔记本模式 (Notebooks)

用户自定义的知识分类容器，支持：
- 创建、编辑、删除笔记本
- 笔记本间节点移动

---

### 4. 查询面板系统 (Query Panel)

右侧常驻查询面板，支持强大的信息检索能力：

| 特性 | 说明 |
|------|------|
| **多查询块** | 最多 3 个并行查询块，方便对比交叉引用 |
| **实时搜索** | 基于内容和标签的语义化节点搜索 |
| **可调宽度** | 面板宽度 320-600px 可调，设置自动持久化 |
| **自然语言** | v3.5+ 支持 AI 解析自然语言查询 |

#### 查询条件类型

| 条件类型 | 说明 | 示例 |
|----------|------|------|
| `tag` | 按标签筛选 | 包含 #任务 标签 |
| `field` | 按字段筛选 | 状态 = "进行中" |
| `keyword` | 关键词搜索 | 内容包含 "会议" |
| `ancestor` | 按祖先节点筛选 | 在某个节点下 |
| `date` | 按日期筛选 | 最近 7 天创建 |

---

### 5. 搜索节点系统 (Search Node)

将查询条件保存为节点，实现：

- **持久化搜索** - 查询条件作为节点存储，支持复用
- **动态结果** - 搜索节点展开时实时计算匹配节点
- **自然语言** - AI 解析自然语言生成结构化查询

---

### 6. AI 指令节点系统 (Command Node)

强大的 AI 助手功能，支持自定义 Prompt 和上下文筛选：

#### 上下文筛选条件

| 筛选维度 | 说明 |
|----------|------|
| `supertagIds` | 按超级标签筛选输入节点 |
| `dateRange` | 按时间范围筛选 |
| `ancestorId` | 按节点路径筛选 |
| `depth` | 子节点遍历深度 |

#### 预设模版分类

| 分类 | 说明 |
|------|------|
| `productivity` | 生产力工具（日程规划、任务分解） |
| `analysis` | 分析工具（数据分析、趋势识别） |
| `creative` | 创意工具（头脑风暴、内容生成） |
| `summary` | 总结工具（会议纪要、周报生成） |

### 7. 智能捕获系统 (Smart Capture) - v3.5+

AI 驱动的智能笔记捕获功能：

| 特性 | 说明 |
|------|------|
| **流式输出** | SSE 流式返回解析结果，实时反馈 |

| **字段提取** | 智能提取结构化字段值 |
| **置信度** | 返回 AI 置信度，支持人工校验 |

### 8. 标签聚焦视图 (Pinned View) - v3.6+

为超级标签提供专属聚焦页面，支持声明式配置：

#### 布局类型

| 布局 | 说明 | 适用场景 |
|------|------|----------|

#### 组件挂载

| 组件类型 | 说明 |

| `ai-aggregation` | AI 聚合摘要组件（如站会纪要） |
| `stats-bar` | 统计信息栏（计数、求和、平均值） |
| `custom` | 自定义组件 |

#### 数据流转规则

- **Quick Capture** - 快速添加节点并设置默认字段值
- **Drag & Drop** - 拖拽卡片自动更新字段值

## 📊 API 端点一览

| 模块 | 端点 | 说明 |
|------|------|------|
| 节点 | `GET/POST/PUT/DELETE /api/nodes` | 节点 CRUD |
| 节点树 | `GET /api/nodes/tree` | 获取完整节点树 |
| 节点同步 | `POST /api/nodes/sync` | 增量同步 |
| 功能标签 | `GET /api/supertags` | 只读标签查询 |
| 查询 | `POST /api/queries/search` | 执行搜索查询 |
| 用户 | `GET/POST /api/users` | 用户管理 |
| AI 聚合 | `POST /api/ai/aggregate` | AI 智能聚合 |

## 🚀 快速开始

### Docker 一键部署

```bash
# 克隆项目
git clone <repository-url>
cd Knowledge_Node

# 构建并启动

# 查看日志
docker-compose logs -f
```
